SCSS Bilan Stillashtirish

SCSS o'zgaruvchilar, mixinlar va nesting yordamida chiroyli va saqlanishi oson stillarni yarating

üìö SCSS Nima?

SCSS (Sassy CSS) - bu CSS'ni o'zgaruvchilar, nesting va mixinlar kabi kuchli funksiyalar bilan kengaytiradigan preprocessordir. U stillarni tashkil qilingan, qayta foydalanish mumkin va saqlashga oson qiladi.

Nega SCSS ishlatish kerak?

  • O'zgaruvchilar - Ranglar, o'lchamlar va qiymatlarni qayta ishlatish uchun saqlash
  • Nesting - Tozaroq, tartibli CSS yozish
  • Mixinlar - Qayta foydalaniladigan stil patternlarini yaratish
  • Funksiyalar - Hisob-kitoblar va transformatsiyalar bajarish

üé® O'zgaruvchilarni Sozlash

O'zgaruvchilar sizning stillar jadvalingizda ishlatiladigan qiymatlarni saqlash imkonini beradi. Bu doimiy ranglar va intervallarni saqlash hamda global o'zgarishlarni tez amalga oshirishni osonlashtiradi.

SCSS - Puzzle O'yini Uchun O'zgaruvchilar 
// Color palette for the puzzle game
$primary-color: #6366f1;      // Indigo - main brand color
$secondary-color: #8b5cf6;    // Purple - accent color
$success-color: #10b981;      // Green - for win state
$text-color: #1f2937;         // Dark gray for text
$bg-color: #f9fafb;           // Light background

// Spacing scale
$spacing-sm: 0.5rem;   // 8px
$spacing-md: 1rem;     // 16px
$spacing-lg: 1.5rem;   // 24px
$spacing-xl: 2rem;     // 32px

// Border radius
$radius: 0.5rem;       // 8px
$radius-lg: 1rem;      // 16px

// Shadows
$shadow: 0 4px 6px rgba(0, 0, 0, 0.1);
$shadow-lg: 0 10px 15px rgba(0, 0, 0, 0.1);

// Transitions
$transition: 300ms ease;
Nomlash Qoidalari
Faqat qiymatni emas, balki maqsadni tushuntiruvchi tavsiflovchi o'zgaruvchi nomlaridan foydalaning. Masalan, $primary-color $bluedan yaxshiroqdir, chunki siz keyinchalik bir xil maqsadni saqlab qolgan holda rangni o'zgartirishingiz mumkin.

üî≤ Puzzle Taxtasini Stillashtirish

Puzzle taxtasi plitalarni mukammal kvadrat tarmoqda joylashtirish uchun CSS Grid'dan foydalanadi. SCSS nesting barcha tegishli stillarni bir joyda tartibga solishga yordam beradi.

SCSS - Puzzle Taxtasi Stillari 
.puzzle-board {
  // Use CSS Grid for perfect tile layout
  display: grid;
  grid-template-columns: repeat(3, 1fr); // 3 equal columns
  gap: $spacing-sm;                      // Space between tiles
  
  // Size and spacing
  max-width: 450px;
  aspect-ratio: 1;        // Make it a perfect square
  margin: 0 auto;         // Center horizontally
  padding: $spacing-sm;
  
  // Visual styling
  background: $bg-color;
  border-radius: $radius-lg;
  box-shadow: $shadow;
  
  // Responsive design - smaller on mobile
  @media (max-width: 768px) {
    max-width: 100%;
    gap: 0.25rem;
  }
}

üß© Puzzle Plitalarini Stillashtirish

Har bir plita fon rasmining bir qismini ko'rsatishi va foydalanuvchi o'zaro ta'siriga silliq o'tishlar va hover effektlari bilan javob berishi kerak.

SCSS - Puzzle Plita Stillari 
.puzzle-tile {
  // Make sure the background image covers the entire tile
  background-size: 300%;      // For 3x3 grid (100% per tile)
  background-repeat: no-repeat;
  background-color: #f3f4f6;  // Fallback color
  
  // Visual effects
  border-radius: $radius;
  cursor: grab;               // Show it's draggable
  transition: transform $transition, box-shadow $transition;
  box-shadow: 0 1px 2px rgba(0, 0, 0, 0.05);
  
  // Hover effect - tile lifts up slightly
  &:hover {
    transform: scale(1.05);
    box-shadow: $shadow;
  }
  
  // Active/dragging state
  &:active {
    cursor: grabbing;
  }
  
  // When being dragged
  &.dragging {
    opacity: 0.5;
    transform: scale(0.95);
  }
}

// For a 4x4 puzzle, use 400% background size
.puzzle-board.grid-4x4 .puzzle-tile {
  background-size: 400%;
}
Background-Size Tushunish

3√ó3 tarmoq uchun biz background-size: 300% ishlatamiz, chunki:

  • Har bir plita rasmning kengligi va balandligining 1/3 qismini ko'rsatadi
  • 300% degani fon plitadan 3√ó kattaroqdir
  • Keyin uni background-position yordamida joylaymiz

‚ú® Qayta Foydalaniladigan Mixinlar Yaratish

Mixinlar - bu istalgan joyda qo'shishingiz mumkin bo'lgan qayta foydalaniladigan stil bloklaridir. Ular kontentni markazlashtirish yoki tugmalar yaratish kabi umumiy patternlar uchun juda mos keladi.

SCSS - Foydali Mixinlar 
// Mixin for centering content with flexbox
@mixin flex-center {
  display: flex;
  justify-content: center;
  align-items: center;
}

// Mixin for consistent button styles
@mixin button($bg-color) {
  display: inline-flex;
  align-items: center;
  padding: 0.75rem 1.5rem;
  background: $bg-color;
  color: white;
  border: none;
  border-radius: $radius;
  font-weight: 600;
  cursor: pointer;
  transition: all $transition;
  
  &:hover {
    transform: translateY(-2px);
    box-shadow: $shadow;
  }
  
  &:active {
    transform: translateY(0);
  }
}

// Using the mixins
.puzzle-controls {
  @include flex-center;
  gap: $spacing-md;
  margin-bottom: $spacing-lg;
  
  button {
    @include button($primary-color);
  }
}

üéâ G'alaba Xabarini Stillashtirish

O'yinchi puzzleni yechganda, biz silliq animatsiyalar bilan tabrik xabarini ko'rsatamiz.

SCSS - Animatsiya Bilan G'alaba Xabari 
// Win message overlay
.win-message {
  position: fixed;
  top: 50%;
  left: 50%;
  transform: translate(-50%, -50%) scale(0);  // Start hidden
  background: white;
  padding: 3rem;
  border-radius: $radius-lg;
  box-shadow: $shadow-lg;
  text-align: center;
  z-index: 2000;
  
  // Show with pop-in animation
  &.show {
    animation: popIn 0.5s ease forwards;
  }
  
  // Animated emoji
  .win-emoji {
    font-size: 4rem;
    margin-bottom: $spacing-md;
    animation: bounce 0.6s ease infinite;
  }
  
  .win-title {
    color: $success-color;
    font-size: 2rem;
    margin-bottom: $spacing-md;
  }
}

// Pop-in animation
@keyframes popIn {
  0% {
    transform: translate(-50%, -50%) scale(0);
  }
  50% {
    transform: translate(-50%, -50%) scale(1.1);
  }
  100% {
    transform: translate(-50%, -50%) scale(1);
  }
}

// Bounce animation
@keyframes bounce {
  0%, 100% {
    transform: translateY(0);
  }
  50% {
    transform: translateY(-10px);
  }
}

To'liq Puzzle O'yini Stillari

Mana puzzle o'yininiz uchun barcha stillarni o'z ichiga olgan to'liq SCSS fayli. Buni puzzle.scss sifatida saqlang va CSS'ga kompilyatsiya qiling.

// Mixins

  display: flex;
  justify-content: center;
  align-items: center;

// Controls

// Board

  gap: 0.5rem;

  margin: 0 auto;
}

// Tiles
