HTML Strukturasini Yaratish

Puzzle o'yini uchun semantik HTML asosini yaratishni o'rganing

üìö Nimani O'rganasiz

Ushbu darsda biz puzzle o'yinimiz uchun HTML strukturasini yaratamiz. HTML (HyperText Markup Language) har bir veb-sahifaning asosidir‚Äîu foydalanuvchilar ko'radigan va muloqot qiladigan struktura va kontentni belgilaydi.

Yaxshi HTML strukturasi uy uchun mustahkam poydevorga o'xshaydi. U quyidagicha bo'lishi kerak:

  • Semantik - Kontentni tavsiflovchi ma'noli teglardan foydalanadi
  • Accessible - Hamma, shu jumladan nogironligi bo'lgan odamlar tomonidan ishlatilishi mumkin
  • Tartibli - Tushunish va saqlashga oson

üèóÔ∏è O'yin Taxtasi Konteyneri

Bizning puzzle o'yinimizga barcha qismlarni ushlab turish uchun konteyner kerak. Buni butun puzzleni ushlab turadigan rasm ramkasiga o'xshatish mumkin. Biz uni aniqlash uchun maxsus klass nomi bilan <div> elementidan foydalanamiz.

O'yin taxtasi quyidagilarni o'z ichiga oladi:

  • Boshqaruv tugmalari (aralashtirish, qayta boshlash)
  • Puzzle tarmog'i (plitalar joylashadigan joy)
  • G'alaba xabari ko'rsatish
HTML - Asosiy O'yin Strukturasi 
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Puzzle Game</title>
    <link rel="stylesheet" href="styles/main.css">
</head>
<body>
    <!-- Main game container -->
    <div class="demo-container">
        <header class="demo-header">
            <h1>üß© Puzzle Game</h1>
            <p>Rearrange the tiles to complete the puzzle!</p>
        </header>

        <!-- The game board -->
        <div class="puzzle-game">
            <!-- Control buttons -->
            <div class="puzzle-controls">
                <button class="btn btn-primary">Shuffle</button>
                <button class="btn btn-secondary">Restart</button>
            </div>

            <!-- Puzzle grid (tiles will go here) -->
            <div class="puzzle-board" id="puzzleBoard">
                <!-- Tiles will be added by JavaScript -->
            </div>
        </div>
    </div>
</body>
</html>
Asosiy Nuqtalar
  • <meta viewport> tegi o'yinni mobil qurilmalarda responsiv qiladi
  • Yaxshi struktura uchun <header> kabi semantik HTML5 teglaridan foydalanamiz
  • puzzle-board kabi klass nomlari element nima qilishini tavsiflaydi
  • Puzzle taxtasiga ID beramiz, shunda JavaScript uni oson topishi mumkin

  • Asl pozitsiya - Hal qilinganda qayerda bo'lishi kerak
  • Fon rasmi - Rasmning qaysi qismini ko'rsatish

Data Atributlarini Tushunish

data-* atributlari HTML elementlarida ma'lumotlarni saqlash uchun foydalaniladigan maxsus atributlardir. JavaScript bu qiymatlarni o'qiy va o'zgartirishi mumkin. Har birining ma'nosi:

  • data-position - Hozirgi pozitsiya (3√ó3 tarmoq uchun 0-8)
  • data-correct-position - Puzzle hal qilinganda bu plita qayerda bo'lishi kerak
  • draggable="true" - Foydalanuvchiga bu elementni sudrab o'tkazish imkonini beradi

‚ôø Accessibility Qilish

Accessibility degani o'yiningizdagi hamma, shu jumladan ekran o'quvchilar yoki klaviatura navigatsiyasidan foydalanadigan odamlar foydalanishi mumkinligini ta'minlashdir. Puzzlemizni accessible qilish usullari:

HTML - Accessible Puzzle Strukturasi 
<div class="puzzle-game" role="application" aria-label="Puzzle Game">
    <div class="puzzle-controls">
        <button class="btn btn-primary" 
                id="shuffleBtn"
                aria-label="Shuffle puzzle pieces">
            Shuffle
        </button>
        <button class="btn btn-secondary" 
                id="restartBtn"
                aria-label="Restart puzzle game">
            Restart
        </button>
    </div>

    <div class="puzzle-board" 
         id="puzzleBoard"
         role="grid"
         aria-label="Puzzle board with 9 tiles">
        <div class="puzzle-tile" 
             role="gridcell"
             tabindex="0"
             aria-label="Puzzle piece 1 of 9"
             data-position="0" 
             data-correct-position="0"
             draggable="true">
        </div>
        <!-- More tiles... -->
    </div>
</div>
Accessibility Funksiyalari Tushuntirildi
  • role="application" - Ekran o'quvchilarga bu interaktiv ilova ekanligini bildiradi
  • aria-label - Ekran o'quvchilar uchun matn tavsifini beradi
  • tabindex="0" - Elementni klaviatura orqali fokuslash mumkin qiladi
  • role="grid" - Puzzle taxtasi strukturasini tasvirlaydi
